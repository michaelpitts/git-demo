# Git Demo Project

A simple, static HTML page that provides an introduction to Git version control system.

## Overview

This project contains a single `index.html` file that serves as an educational resource about Git. It covers:

- **History of Git** - How Linus Torvalds created Git in 2005 for Linux kernel development
- **Basic Principles of Git** - Core concepts including distributed version control, snapshots, branching, data integrity, and the staging area
- **Common Git Commands** - A quick reference of essential Git commands

## Features

- Clean, responsive design
- No external dependencies
- Mobile-friendly layout
- Git-themed color scheme (using Git's signature orange #f05032)

## Usage

Simply open `public/index.html` in any web browser to view the content.
